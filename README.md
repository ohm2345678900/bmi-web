# BMI Calculator Web

โปรเจกต์นี้เป็นเว็บแอปพลิเคชันสำหรับคำนวณค่า BMI (Body Mass Index)  
จัดทำขึ้นเพื่อฝึกการทำงานเป็นทีมด้วย Git และ GitHub

---

## Features
- กรอกส่วนสูง (cm) และน้ำหนัก (kg)
- คำนวณค่า BMI
- แสดงผลลัพธ์พร้อมสถานะ (ผอม / ปกติ / อ้วน)
- แยกงานเป็น HTML, CSS, JavaScript และ QA

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Git & GitHub
